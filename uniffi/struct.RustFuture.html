Expand description

Future that the foreign code is awaiting

RustFuture is always stored inside a Pin<Arc<>>. The Arc<> allows it to be shared between wakers and Pin<> signals that it must not move, since this would break any self-references in the future.

Wake up soon and poll our future.

This method ensures that a call to do_wake() is scheduled. Only one call will be scheduled at any time, even if wake_soon called multiple times from multiple threads.
